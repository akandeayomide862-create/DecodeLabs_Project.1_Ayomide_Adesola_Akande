# 📊 E-Commerce Sales Data Cleaning and Preparation

## 📌 Overview
As part of the **DecodeLabs Data Analytics Internship Program**, this project involved cleaning and preparing an e-commerce sales dataset using Microsoft Excel. The dataset consisted of 1,200 records and 14 fields containing information related to customer purchases, transactions, payment methods, and sales activities.

The primary aim of this task was to improve the quality and reliability of the data before proceeding to exploratory data analysis and visualization.

## 🎯 Objectives
- Identify and eliminate duplicate records.
- Detect and address missing values.
- Verify and correct data formats where necessary.
- Prepare the dataset for further analysis and reporting.

## 📂 Dataset Description
The dataset, provided by DecodeLabs, contained transactional information including:
- Order ID
- Date
- CustomerID
- Product
- Quantity
- Unit Price
- Shipping Address
- Payment Method
- Order Status
- Tracking Number
- Items In Cart
- Coupon Code
- Referral Source
- Total Price

## 🛠️ Tools and Features Used
- Remove Duplicates
- Find and Replace
- Formatting Tools
- Excel Functions

## 🧹 Data Cleaning Process

### 1. Duplicate Record Check
A comprehensive review of the dataset was conducted using Excel’s **Remove Duplicates** feature. After checking all records, no duplicate entries were found.

### 2. Handling Missing Values
Missing values were identified using filters and Find & Replace. The **CouponCode** column contained missing values, which were replaced with **"UNKNOWN"** to avoid data loss. A total of **309 entries** were updated.

### 3. Data Type Formatting
Several formatting improvements were applied to ensure consistency:
- Date column was converted to proper date format.
- Numerical values were formatted as Number/Currency.
- Text fields were standardized for consistency.

### 4. Verification of Calculated Values
The **Total Price** column was validated by comparing it with **Quantity × Unit Price** to ensure accuracy and consistency.

## 📌 Skills Demonstrated
- Data Cleaning and Preparation
- Data Quality Validation
- Excel Spreadsheet Management
- Data Formatting and Standardization
- Use of Excel Functions

## 🏁 Conclusion
The cleaning process improved the overall quality and reliability of the dataset by addressing missing values, ensuring consistency, and validating calculations. This made the dataset suitable for further analysis and visualization, supporting accurate business insights.
